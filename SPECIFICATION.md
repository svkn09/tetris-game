# Tetris Game - 仕様書

## 1. ゲーム概要
このテトリスゲームは、落ちてくるテトリミノを操作し、横一列にブロックを揃えて消していくパズルゲームです。ラインを消すことでスコアを獲得し、レベルが上がると落下速度が速くなります。ゲームボードの最上部までブロックが積み上がるとゲームオーバーとなります。

## 2. 操作方法
キーボードを使用してテトリミノを操作します。

| キー          | 操作内容                                 |
| :------------ | :--------------------------------------- |
| **← (左矢印)** | 現在のテトリミノを左に1マス移動          |
| **→ (右矢印)** | 現在のテトリミノを右に1マス移動          |
| **↑ (上矢印)** | 現在のテトリミノを時計回りに90度回転     |
| **↓ (下矢印)** | 現在のテトリミノを1マス下に素早く落下 (ソフトドロップ) |
| **Space**     | 現在のテトリミノを一瞬で最下部まで落下 (ハードドロップ) |
| **Z**         | 現在のテトリミノをホールド（一時保管）する。ホールド中のテトリミノがある場合は入れ替える。1回の落下中に1度のみ使用可能。 |
| **P**         | ゲームの一時停止/再開を切り替える        |

## 3. ゲームルール

### 3.1. テトリミノの生成と落下
- ゲーム開始時、およびテトリミノが固定されるたびに、ランダムなテトリミノがゲームボード上部に生成されます。
- テトリミノは一定速度で自動的に落下します。

### 3.2. ライン消去
- 横一列にブロックが隙間なく揃うと、そのラインが消去されます。
- 消去されたラインの上のブロックは、重力に従って下に落下します。

### 3.3. ゲームオーバー
- 新しく生成されたテトリミノが、生成された時点で既存のブロックと重なり、移動できない場合、ゲームオーバーとなります。

## 4. スコア計算ルール
ラインを消去することでスコアが加算されます。一度に消去するライン数が多いほど、高得点が得られます。レベルが高いほど、獲得スコアに倍率がかかります。

| 消去ライン数 | 基本スコア |
| :----------- | :--------- |
| 1ライン      | 100点      |
| 2ライン      | 300点      |
| 3ライン      | 500点      |
| 4ライン      | 800点      |

**最終スコア = 基本スコア × 現在のレベル**

- ソフトドロップ: 1マス落下ごとに1点加算されます。
- ハードドロップ: 落下したマス数 × 2点加算されます。

## 5. レベルアップ条件
消去したライン数の合計が10ラインに達するごとにレベルが1上がります。

- **レベルアップ効果**: テトリミノの落下速度が速くなります。具体的には、レベル1では1000ms（1秒）ごとに落下しますが、レベルが1上がるごとに落下間隔が100ms短縮されます（最低100ms）。

## 6. UI機能
- **SCORE**: 現在のスコアを表示します。
- **LEVEL**: 現在のゲームレベルを表示します。
- **LINES**: これまでに消去したラインの合計数を表示します。
- **NEXT**: 次に落下するテトリミノの形状を表示します。
- **HOLD**: ホールドしているテトリミノの形状を表示します。
- **STATUS**: ゲームの状態（READY, PLAYING, PAUSED, GAME OVER）を表示します。

## 8. 設計方針
- **可読性**: 初心者でも理解しやすいよう、シンプルで分かりやすいコード構造を心がけました。
- **モジュール性**: ゲームロジックを適切に分割し、各機能が独立して動作するように設計しました。
- **コメント**: コードの意図や複雑なロジックには適宜コメントを付与し、学習教材としての価値を高めています。
